# Chip8-Plus-Plus
A chip 8 emulator written in C++ using SFML

#### Dependencies

	- sfml
	- gcc
	
#### Building:

Just type 

	make 
	
in the root of the repo

#### Running:

Type 

	./chip8 ./roms/[ROM] 

where [ROM] is the game you want to run
